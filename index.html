Frequently Asked Questions

1. What is the difference between var, let and const?

var, let and const are used to declare a variable in javascript. var is function scoped and let, const are block scoped. we can redeclare and reassign variable created with var. we can reassign variable created with let but we can not redeclare or reassign value to variable created with const.

2. Difference between map, filter and foreach method.

map, filter and foreach all are array methods that are used to loop through and perform certain action on an array. The map method allow us to run a function on every element on an array and return them into a new array. The filter method loop through an array and perform a conditional check on each element. If the element pass the condition then it'll be returned into a new array. The foreach method also used to loop through an array. But it does not return the elements into new array.

3. Arrow function and difference with normal function.

Arrow function is a simple version of regular function. the function is written by declaring a variable then a () following by arrow sign => and then bracket. This allows us to write function in easy and simple way.

4. How promise work?

Promise is a object the produce a single value now or sometimes in the future. Promise mainly return two thing, resolve and reject. if the promise resolves then it will return the desired output otherwise if it gets rejected the it'll throw error.

5. How closures work in JavaScript?

A closure is a function that has access to its parent scope after parent function closed. Normally a variable defined inside a function dies when the function returns or finishes. But using a function inside a function we can still use the variable and all the parent functions scope. This is called closure.
